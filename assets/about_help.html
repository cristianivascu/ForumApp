About this application

The Informatics Forum is the main building of the School of Informatics at the University of Edinburgh. Every year it plays host to numerous significant events, with visitors attending from all over the world. This application was developed to assist visitors who come to the Informatics Forum in planning their trip and finding particular places nearby. Please read the user documentation to see a detailed explanation of the features and information on how to use them. Below you will find the solutions to some common problems.

Problem, meet solution

  1. I have cleared all the filters, but some entries are still missing from the list.
    While the filters were applied, you might have deleted an entry. Press "Restore Deleted" from the menu and you should see all your entries.

  2. I have restored the deleted entries, but some are still missing.
    Is there any filter applied? Make sure you have cleared all the filters.

  3. I am trying to add an entry but it doesn't appear in the list.
    This happens when a filter is set and the entry you are trying to add doesn't meet the criteria of the filter. Clear all the filters and you should see your entry.

  4. I see some question marks in the description of my entry instead of other characters
    You have used either the _ or | character in your description. Unfortunately, it is not possible to use these characters in this version and these will be replaced automatically.

  5. After applying a filter, I get results where the information used by the filter is not available. Is this normal?
    Yes, this is how the application is designed to work. A more detailed explanation is available in the user documentation.
